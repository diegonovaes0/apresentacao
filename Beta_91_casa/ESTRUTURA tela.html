(em vez de 11-12px) Informações auxiliares/badges: 9px (em vez de 11-12px) Componentes Interativos Altura de botões: 22-26px (em vez de 32-36px) Padding interno de botões: 3-4px vertical, 8-10px horizontal Tamanho de ícones: 10-12px (em vez de 14-16px) Altura de inputs/selects: 24px (em vez de 32-36px) Cards e Containers Card de host: 160-180px largura × 85-96px altura Card de playbook: padding de 6px (em vez de 8-12px) Card de execução: padding de 8-10px (em vez de 12-16px) Espaçamento entre cards: 8px (em vez de 10-15px) Elementos Visuais Indicadores de status/barras laterais: 3px largura (em vez de 4-5px) Barras de progresso: 3px altura (em vez de 4-5px) Altura de headers: 26-30px (em vez de 40-50px) Badges/contadores: padding 1-2px vertical, 5-6px horizontal Espaços para Scroll Altura de áreas de rolagem: 250-350px (em vez de 400-600px) Largura de barras de rolagem: 4px (em vez de 6-8px) Dicas para Solicitar Design Compacto Quando solicitar a outra IA, use frases como: "Preciso de uma interface ultra-compacta onde cada elemento tenha no máximo 70-80% do tamanho padrão" "Use fonte de 10-11px para o conteúdo principal e 9px para informações secundárias" "Reduza todo o espaçamento interno (padding) para 6-8px e o espaçamento entre elementos para 8px" "Mantenha os cards de host com aproximadamente 180×96px para maximizar o número visível na tela" "Prefira indicadores visuais sutis (como barras laterais de 3px) em vez de grandes badges ou ícones" "Faça os botões com altura de 22-26px e campos de seleção com 24px para economizar espaço vertical"
